Detailed Description

Provides general-purpose utility functions for the VoronoiMeshMaker library.

This file defines utility functions used throughout the VoronoiMeshMaker library, including functions for printing lines, manipulating strings, handling file paths, and converting smart pointers. These functions are part of the "misc" group of utility components.
