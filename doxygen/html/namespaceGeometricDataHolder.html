Storage map for data.

The data map stores values of various types, accessible via string keys. Supported types include:

  • int
  • Real (defined in VoronoiMeshMaker type.h)
  • std::string
  • gtp::Point2D, gtp::Vector2D, gtp::Point3D, gtp::Vector3D
  • std::list<int>, std::list<Real>

Each entry in the map is associated with a key that allows for easy retrieval and modification of the stored values.
