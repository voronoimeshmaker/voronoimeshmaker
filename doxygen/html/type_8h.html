Detailed Description

Defines common geometric and numeric types used in the VoronoiMeshMaker library.

This file defines key types used in computational geometry and numerical calculations. Types such as points, segments, vectors, and polygons in 2D and 3D are defined here, along with smart pointers for memory management.
