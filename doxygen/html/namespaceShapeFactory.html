Creates a shape based on its ClassID.

Finds the associated shape creation function in the registry using the provided ClassID and returns a shared pointer to the newly created shape, initialized with the provided data.

Parameters
classIDThe shared ID of the shape to create.
dataThe geometric data used to initialize the shape.
Returns
std::shared_ptr<Shape> A shared pointer to the created shape.
Exceptions
std::runtime_errorif the ClassID is not found in the registry.

This method looks up the provided ClassID in the shape registry to find the corresponding creation function. If found, it uses this function to create and return an instance of the shape.

Registers a shape type in the factory using its ClassID.

This function associates a ClassID with a function that creates an instance of the shape when provided with a GeometricDataHolder.
