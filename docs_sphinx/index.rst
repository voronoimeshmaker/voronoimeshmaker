Project Overview
================

**VoronoiMeshMaker** is a library designed for generating 2D and 3D Voronoi meshes.
It is primarily used in computational geometry and can be integrated into projects that use finite volumes in the discretization of differential equations.

This documentation will cover:
- How to install the library
- API reference for key classes and methods
- Example usages and applications
