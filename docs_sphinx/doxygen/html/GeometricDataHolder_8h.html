Detailed Description

Defines the GeometricDataHolder class and the DataTraits for handling multiple data types. This class uses std::variant to store different types and allows access through string keys. The DataTraits structure provides type identification for the supported data types.

This file is part of the "parameters" group, which defines classes and traits to handle multiple types of parameters for computational geometry and other operations.
