Detailed Description

Defines the DataExporter class responsible for exporting geometric data in various formats.

This file is part of the VoronoiMeshMaker library, which is used for the generation of Voronoi meshes and related operations. It allows flexible exportation of geometric data such as sites, shapes, Delaunay triangles, and Voronoi diagrams to external formats like VTK and HDF5.
